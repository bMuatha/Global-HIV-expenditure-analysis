# Global HIV Expenditure Analytics Dashboard
#introduction 

This project is licensed under the [Creative Commons Attribution 4.0 International License](https://creativecommons.org/licenses/by/4.0/).

This project analyzes HIV expenditure trends using global datasets from 
Our World in Data. It includes:

- R script for data import and cleaning  
- Power BI data model  
- DAX measures for KPIs  
- Interactive visualization dashboard  

## Data Source
Our World in Data – HIV Expenditure  
https://ourworldindata.org/hiv-expenditure

## Tools Used
- R (jsonlite, dplyr)
- Power BI
- DAX
- Git & GitHub

## Folder Structure
- /data — Raw and cleaned datasets 
- /r-scripts — R cleaning scripts  
- /powerbi — Final Power BI dashboard (.pbix)  
- /dax — All DAX measures used
- /visuals — Screenshots for portfolio display  

## Dashboard Highlights
- Total HIV Expenditure 
- Number of countries reporting
- latest year reported
